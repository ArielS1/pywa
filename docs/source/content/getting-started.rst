================================

Everything in pywa start from the :class:`pywa.WhatsApp` Client. The client responsible for all the communication with the WhatsApp servers and also to register the callbacks for the incoming messages.

First, you need **Access token** and **Phone number ID** to start using the API.
    - Follow `this section <setup.html>`_ to get the credentials.
